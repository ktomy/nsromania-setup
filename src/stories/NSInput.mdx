## NSInput
A simple input field implementing all the props from the Material UI library. 

[TextField API documentation](https://material-ui.com/api/text-field/)

In the future when we will implement Zod and react-hook-form, we will have to add the props from those libraries as well.



### Contributing
This is a main component for user input. It can be used for text input, number input, password input, etc.
It is reused accross the project, this is why when contributing to this component it should be well documented.
